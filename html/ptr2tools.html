TUTORIAL
TO FREE MINGW
Run "cd /".
From here you can CD to the C (or any) drive by doing "cd c".
Note that you will have to do this everytime you start up MINGW.
COMPILING AND SETUP
  1. Install MSYS2 from https://www.msys2.org/ and open the MINGW64 console.
  2. Use the cd command to direct where you want the installation to go.
  3. Download the make.sh in the releases page and place it into the directory (do "cd ." if you were in the dir you put it in)
  4. Use the command "sh make.sh"
  5. Say yes to all packages it prompts.

Once this finishes, you should have PTR2Tools installed in MINGW64. If you wish to check for updates, run the make.sh again.

DOCUMENTATION
PTR2INT
list [intfile]
Description
Prints a list of contents of an INT file.
Aliases: l, ls
Arguments
intfile: The INT file. It's best to have this in your cd directory.
extract [intfile] [folder]
Description
Extracts files from an INT into a folder
Aliases: e, ex, x
Arguments
intfile: The INT file to extract from. It's best to have this in your cd directory.
folder: The folder to output all files to.
create [intfile] [folder]
Description
Creates an INT file from a folder.
Aliases: c, i, make, pack
Arguments
intfile: The INT file to create.
folder: The folder to create the INT from.
optimize [folder] [out-neworderfile]
Description
I do not recommend using this.
Attempts to optimize the _order.txt file of a folder to achieve lzss compression.
Alias: optimise
Arguments
folder: The folder created from the extract command that wishes to be optimized.
out-neworderfile: The new order file to be created.
